Pour installer le site et le faire fonctionner : 

	* Faire marcher faxebook : aller dans votre fichier hosts.conf qui se trouve dans le system32
et rajouter la ligne sandbox.com 127.0.0.1
Dorénavant, vous appelerez l'URL avec sandbox.com au lieu de 127.0.0.1
(Sinon facebook ne fonctionne pas en local)

	*Pour commencer à utiliser l'application, après dézippage du projet, tapez 
sandbox.com/PPIL/cakephp

Enjoy :)
